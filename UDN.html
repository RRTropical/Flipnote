SNAPSHOTS (Yeah, totally)

10/29/22:

1} Fixed issue where when you edit a note the context menu pops up when you finish

This is what the context menu is for the people who don't know:



2} Added "Welcome In!" feature which means that in the editing menu, if you put a "+"" before your message it will actually add that to the existing message in your note. For example, if I had "I am", and edited it and put "+ Flipdev!" then the note would be edited to "I am Flipdev!" and if I had a note saying "I am Flipdev", edited it and put "+!" (no space), it would say "I am Flipdev!"


11/2/22:

1} Added "Eviction" feature which means that in the editing menu, if you put a "-" before your message it will get rid of the first occurence of that message in your note. For example, if I had "I am Flipdev", and edited it and put "-Flipdev" then the note would be edited to "I am"


11/5/22:

1} changed editing system to make it more "user friendly". This means you'll have to say goodbye to the +/- feature and the old nasty editing dialog and say hello to the new system and Adialog!

2} Changed Flipnote history colors to make it more readable
