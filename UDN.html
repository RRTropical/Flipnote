2} New Feature: Added "Welcome In!" feature which means that in the editing menu, if you put a "+"" before your message it will actually add that to the existing message in your note. For example, if I had "I am", and edited it and put "+ Flipdev!" then the note would be edited to "I am Flipdev!" and if I had a note saying "I am Flipdev", edited it and put "+!" (no space), it would say "I am Flipdev!".


11/2/22:

1} New Feature: Added "Eviction" feature which means that in the editing menu, if you put a "-" before your message it will get rid of the first occurence of that message in your note. For example, if I had "I am Flipdev", and edited it and put "-Flipdev" then the note would be edited to "I am".
